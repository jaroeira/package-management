Description Package Management

This demo project was created using the MEAN Stack. (MongoDB - Express.js - Angular - Node.js)

- Frontend => Angular: A Package Feature ngModule contains all the logic related to the package management frontend.

Data access and state management are provided by an observable service which is injected into the components where it is needed.

- Frontend Interface: View Packages: Filter by Type, Suported Devices and Version. From the list the user can edit or delete a package.

- Frontend Interface: Add Packages: Add a new package containing a File and some data.

- Backend => Node.js - Express.js: A rest-api created with Node.js and Express provides the necessary endpoints to create, get, update and delete packages.

A package consists of a file and some data. (title, type, version, supported devices).

- Database => MongoDB. A noSQL database.
